# passman

Password manager for Linux, written in Python 3.  Utilizes gpg and doesn't require third party libraries (yet)


## Dependencies

### GPG
 `sudo apt-get install gpg`

### Python 3
 `sudo apt-get install python3`
 
 *should be setup in a python virtualenv*
 - `python3 -m venv env`
